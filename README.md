# 🤖 Previsão de Evasão Escolar com Machine Learning

## 📌 Sobre o projeto

Este projeto utiliza técnicas de Machine Learning para prever a taxa de evasão escolar com base em dados educacionais.

O dataset foi obtido da plataforma Kaggle e contém informações sobre financiamento, desempenho e características sociais das escolas.

---

## 📊 Dataset

- Nome: Education Inequality Data
- Fonte: Kaggle

---

## 🎯 Objetivo

Prever a variável:

- dropout_rate_percent (taxa de evasão escolar)

---

## 🧠 Etapas do projeto

- Coleta de dados
- Análise exploratória (EDA)
- Pré-processamento
- Treinamento de modelo
- Validação

---

## ⚙️ Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

---

## 📈 Modelos utilizados

- Árvore de decisão
- GridSearchCV para otimização

---

## 📄 Resultados

O modelo foi avaliado utilizando:

- R²
- RMSE
- MAE
- Classification Report
